1. Eth0 - NAT, used by vagrant for SSH
   Eth1 - Private_Network : Management purpose of Openstack
   Eth2 - Public_Network
2. Remove unnecessary bridges on your host
   ip link set br100 down
   brctl delbr br10
3. sudo apt-get install brdige_utils
   Config your network to something similar for Vagrant to work
   /etc/network/interfaces

   auto lo
   iface lo inet loopback

   auto br0
   iface br0 inet dhcp
   bridge_ports enp0s31f6
   bridge_stp off

   sudo /etc/init.d/networking restart

   vagrant plugin install vagrant-cachier
